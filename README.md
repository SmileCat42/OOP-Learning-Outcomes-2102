<h1 align="center"><b>COS2102</b></h1>
<h3 align="center">Object - Oriented Programming การเขียนโปรแกรมเชิงอ็อบเจกต์</h3>
<br><br>
<h2 style="text-indent: 2em; text-decoration: underline;">เนื้อหาที่ได้รับ</h2>
1. เข้าใจวิธีใช้งาน Class และการสร้าง Object ว่าทั้ง 2 มีความสัมพันธ์กันอย่างไร นำไปใช้งานในลักษณะไหนได้บ้าง
<br>2. สามารถสร้าง Default Constructor, Constructor และ Destructor ได้ เข้าใจการทำงานลำดับการเกิดและการตาย 
<br>3. สามารถสร้างภาพในหัวได้ว่า Dynamic array คืออะไร มีลักษณะเป็นอย่างไรและมีเงื่อนไขอย่างไร
<br>4. ทราบถึงคำสั่งความสัมพันธ์ friend ทั้งแบบ class และ function ว่ามีประโยชน์และข้อควรระวังอย่างไร
<br>5. เข้าใจความสามารถในการสืบทอดหรือ Inheritance ทั้งแบบทางเดียวและแบบ Multiple Inheritance รวมถึงสิทธ์ในการเข้าถึงคลาสแม่หรือ baseclass ในแบบต่างๆ ไม่ว่าจะเป็นการสืบทอดแบบ private, protected หรือ public ซึ่งสามารถแยกแยะความสามารถในการเข้าถึงข้อมูลของคลาสลูกในแต่ละแบบได้
<br>6. เข้าใจปัญหาของการสืบทอดหลายทาง ซึ่งจะทำให้เวลาเรียกใช้คำสั่งต่างๆจะเกิดความกำกวมได้ง่าย ทำให้สามารถแก้ไขปัญหาตรงจุดนี้ได้ด้วยการ virtual ซึ่งจำเป็นต้องศึกษาถึงวิธีใช้งานและข้อจำกัดของคำสั่งนี้ด้วย
<br>7. สามารถสร้าง Operator Overloading ในแบบต่างๆได้ และสามารถนำไปใช้งานในแต่ละแบบได้จริง
<br>8. นำหลักการ Polymorphism ไปใช้งานโดยใช้ Dynamic binding ได้จริง เข้าใจถึงการทำ Abstract class หรือเพียวฟังก์ชั่น ว่าทำไปเพื่ออะไร นำไปใช้งานอย่างไร
